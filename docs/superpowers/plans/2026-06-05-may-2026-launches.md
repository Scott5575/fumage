# May 2026 Launches Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the seven best men's fragrance launches of May 2026 to the Fumage catalog and publish two articles about them — an evergreen "intensified flanker" essay and a repeatable monthly Dispatch roundup.

**Architecture:** Catalog entries land via a new, isolated `fragrance_ext_2026_may.json` that a small `seed.ts` patch merges into the existing upsert pipeline; search is refreshed via the existing Meilisearch reindex script. Articles are hand-coded Next.js route pages reusing the established Fumage component vocabulary; the roundup is built on a reusable `Dispatch` component so future months are cheap.

**Tech Stack:** Next.js 16 (App Router), Prisma 5 + PostgreSQL (Supabase), Meilisearch, TypeScript, Tailwind, ts-node.

**Spec:** `docs/superpowers/specs/2026-06-05-may-2026-launches-design.md`

```ts
  const raw: RawFragrance[] = JSON.parse(fs.readFileSync(dataPath, "utf-8"));
  console.log(`  Loaded ${raw.length} fragrances from JSON`);

  // ── Merge editorial extension files (fragrance_ext_2026_*.json) ───────────
  // Keeps post-launch editorial additions in their own version-controlled
  // files instead of hand-editing the 1.2 MB main database.
  // Scoped to the "2026_" prefix on purpose: the fragrance_ext_designer_tier*.json
  // files are already baked into fragrance_database.json and must not re-merge.
  const dataDir = path.resolve(__dirname, "data");
  const extFiles = fs
    .readdirSync(dataDir)
    .filter((f) => /^fragrance_ext_2026_.*\.json$/.test(f))
    .sort();
  for (const file of extFiles) {
    const ext: RawFragrance[] = JSON.parse(
      fs.readFileSync(path.join(dataDir, file), "utf-8")
    );
    raw.push(...ext);
    console.log(`  + Merged ${ext.length} fragrances from ${file}`);
  }

  const nameSlugMap = buildNameSlugMap(raw);
```
